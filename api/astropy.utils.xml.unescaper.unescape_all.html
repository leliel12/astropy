unescape_all¶

astropy.utils.xml.unescaper.unescape_all(url) [edit on github][source]¶

Recursively unescape a given URL.

Note

‘&amp;&amp;’ becomes a single ‘&’.

Parameters:

url : str or bytes

URL to unescape.
Returns:

clean_url : str or bytes
