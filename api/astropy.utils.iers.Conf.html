Attributes Documentation

auto_download¶

Enable auto-downloading of the latest IERS data. If set to False then the local IERS-B file will be used by default. (default=True)

auto_max_age¶

Maximum age of predictive data before auto-downloading (days, default=30)

iers_auto_url¶

URL for auto-downloading IERS file data
