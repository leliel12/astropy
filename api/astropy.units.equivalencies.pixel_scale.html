pixel_scale¶

astropy.units.equivalencies.pixel_scale(pixscale)[source] [edit on github]¶

Convert between pixel distances (in units of pix) and angular units, given a particular pixscale.

Parameters:

pixscale : Quantity

The pixel scale either in units of angle/pixel or pixel/angle.
