Pix2Sky_SlantZenithalPerspective¶

class astropy.modeling.projections.Pix2Sky_SlantZenithalPerspective(mu=0.0, phi0=0.0, theta0=90.0, **kwargs) [edit on github][source]¶

Bases: astropy.modeling.projections.Pix2SkyProjection, astropy.modeling.projections.Zenithal

Slant zenithal perspective projection - pixel to sky.

Corresponds to the SZP projection in FITS WCS.

Parameters:

mu : float

Distance from point of projection to center of sphere in spherical radii, μ. Default is 0.

phi0 : float

The longitude φ₀ of the reference point, in degrees. Default is 0°.

theta0 : float

The latitude θ₀ of the reference point, in degrees. Default is 90°.
