SLSQP¶

class astropy.modeling.optimizers.SLSQP [edit on github][source]¶

Bases: astropy.modeling.optimizers.Optimization

Sequential Least Squares Programming optimization algorithm.

The algorithm is described in [R9]. It supports tied and fixed parameters, as well as bounded constraints. Uses scipy.optimize.fmin_slsqp.
