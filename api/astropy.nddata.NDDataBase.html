NDDataBase¶

class astropy.nddata.NDDataBase [edit on github][source]¶

Bases: object

Base metaclass that defines the interface for N-dimensional datasets with associated meta informations used in astropy.

All properties and methods have to be overridden in subclasses. See NDData for a subclass that defines this interface on ndarray-like based data.

Attributes Documentation

data¶

The stored dataset.

mask¶

Mask for the dataset.

Masks should follow the numpy convention that valid data points are marked by False and invalid ones with True.

meta¶

Meta information about the dataset.

Should be dict-like.

uncertainty¶

Uncertainty in the dataset.

Should have an attribute uncertainty_type that defines what kind of uncertainty is stored, such as 'std' for standard deviation or 'var' for variance.

unit¶

Unit for the dataset.

wcs¶

A world coordinate system (WCS) for the dataset.
