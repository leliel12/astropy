PointMeasures¶

class astropy.stats.PointMeasures(p0=0.05, gamma=None, ncp_prior=None) [edit on github][source]¶

Bases: astropy.stats.FitnessFunc

Bayesian blocks fitness for point measures

Parameters:

p0 : float (optional)

False alarm probability, used to compute the prior on (see eq. 21 of Scargle 2012). If gamma is specified, p0 is ignored.

ncp_prior : float (optional)

If specified, use the value of ncp_prior to compute the prior as above, using the definition . If ncp_prior is specified, gamma and p0 are ignored.
