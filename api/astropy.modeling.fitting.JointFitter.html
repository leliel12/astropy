JointFitter¶

class astropy.modeling.fitting.JointFitter(models, jointparameters, initvals) [edit on github][source]¶

Bases: object

Fit models which share a parameter.

For example, fit two gaussians to two data sets but keep the FWHM the same.

Parameters:

models : list

a list of model instances

jointparameters : list

a list of joint parameters

initvals : list

a list of initial values
