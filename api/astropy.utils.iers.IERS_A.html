IERS_A¶

class astropy.utils.iers.IERS_A(data=None, masked=None, names=None, dtype=None, meta=None, copy=True, rows=None, copy_indices=True, **kwargs)[source] [edit on github]¶

Bases: astropy.utils.iers.IERS

IERS Table class targeted to IERS A, provided by USNO.

These include rapid turnaround and predicted times. See http://maia.usno.navy.mil/

Notes

The IERS A file is not part of astropy. It can be downloaded from iers.IERS_A_URL. See iers.__doc__ for instructions on how to use it in Time, etc.
