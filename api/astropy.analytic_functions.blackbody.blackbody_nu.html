blackbody_nu¶

astropy.analytic_functions.blackbody.blackbody_nu(in_x, temperature) [edit on github][source]¶

Calculate blackbody flux per steradian, .

Note

Warning

Output values might contain nan and inf.

temperature : number, array-like, or Quantity

Blackbody temperature. If not a Quantity, it is assumed to be in Kelvin.
Returns:

flux : Quantity
